이어폰이 끊기거나 출력 장치가 바뀌는 순간, 소리가 Mac 스피커로 갑자기 나오는 상황은 생각보다 자주 발생합니다.

회의가 잦거나 공용 공간에서 일하고, 블루투스 장치를 자주 바꾸는 사용자에게 특히 자주 생깁니다.

## 왜 출력 장치 변경이 위험한가

macOS에서는 기본 출력 장치가 바뀌면 소리가 즉시 새 장치로 이동합니다. 문제는:

- 수동 음소거를 누를 시간이 없음
- 장치 변화가 매우 빠르게 일어남(특히 블루투스 불안정 시)
- 회의/공유 공간에서는 실수 허용 범위가 거의 없음

그래서 매번 기억에 의존하는 수동 방식보다 자동 규칙이 훨씬 안정적입니다.

## Default0에서 `출력 전환 시 음소거` 켜기

`출력 전환 시 음소거`는 출력 장치 변경 시점에 먼저 볼륨을 0으로 만드는 Default0의 기본 기능입니다.

### 1단계: Default0 실행

- Default0를 실행하고 메뉴 막대에서 상주 확인
- 정상 동작 상태 확인

### 2단계: 출력 변경 음소거 활성화

- 설정 열기
- `출력 전환 시 음소거` 찾기
- 규칙 켜기

### 3단계: 전환 테스트

- 이어폰/스피커를 수동 전환
- 또는 블루투스 이어폰 재연결
- 전환 순간 자동 음소거 여부 확인

## 더 안전하게: 규칙 2개 추가

다음 두 규칙을 함께 켜면 안정성이 높아집니다.

- `블루투스 해제 시 음소거`
- `잠금 해제 시 음소거`

세 규칙 조합으로 일상적인 돌발 외부 재생 대부분을 막을 수 있습니다.

## 사용 시나리오

### 시나리오 1: 회의 사이 빠른 전환

한 회의를 끝내고 다음 회의로 이동할 때, 웹/앱의 오작동 재생을 막아줍니다.

### 시나리오 2: 이어폰 배터리 부족 또는 연결 불안정

블루투스 끊김으로 스피커로 회수되더라도, 선 음소거로 외부 재생을 막습니다.

### 시나리오 3: 공용 공간 근무

오픈 오피스나 도서관에서는 자동 규칙이 수동 확인보다 훨씬 믿을 수 있습니다.

## FAQ

### 1) `출력 전환 시 음소거`는 무료인가요?

네, 기본 자동 음소거 기능입니다.

### 2) 정상 청취에 영향이 있나요?

아니요. 트리거 시점에만 먼저 음소거되며, 언제든 볼륨을 복구할 수 있습니다.

### 3) 회의 앱 실행 시에도 자동 음소거할 수 있나요?

가능합니다. Pro의 `앱 실행 시 음소거`를 사용하세요.

### 4) 매번 다시 설정해야 하나요?

아니요. 한 번 켜면 계속 유지되고, 필요 시 일시정지/재개할 수 있습니다.
