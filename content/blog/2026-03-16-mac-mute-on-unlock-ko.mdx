맥을 잠금 해제한 직후, 예상치 못한 소리가 스피커로 나오는 상황은 생각보다 자주 발생합니다.

## 왜 잠금 해제 순간이 위험할까

- 이전 볼륨 상태가 그대로 유지됨
- 출력 장치가 바뀌었지만 바로 인지하지 못함
- 다음 회의에 급히 들어가며 확인을 놓침

## Default0에서 `잠금 해제 시 음소거` 켜기

`잠금 해제 시 음소거`는 잠금 해제 직후 시스템 볼륨을 먼저 0으로 만들어 돌발 외부 재생을 막는 기본 기능입니다.

### 1) Default0 실행
- 메뉴 막대에서 Default0 실행
- 백그라운드 실행 상태 확인

### 2) 규칙 활성화
- 설정 열기
- `잠금 해제 시 음소거` 찾기
- 토글 켜기

### 3) 테스트
- 화면 잠금 후 다시 해제
- 볼륨이 자동 음소거됐는지 확인

## 함께 켜면 좋은 규칙 2개

- `출력 전환 시 음소거`
- `블루투스 해제 시 음소거`

## FAQ

### 무료 기능인가요?
네, 기본 기능입니다.

### 계속 음소거 상태로 남나요?
아니요. 트리거 시점에만 먼저 음소거되고, 언제든 해제할 수 있습니다.

### 특정 앱에서만 자동 음소거 가능할까요?
가능합니다. Pro의 `앱 실행 시 음소거`를 사용하면 됩니다.
